# Individual Project

This repository contains work for the Individual Project, with the main PINN work located in `PINN/`.

## Structure
- `PINN/` Physics-Informed Neural Networks project
- `PINN/data/` input data (CSV/TXT ignored in git)
- `PINN/models/` model code (trained checkpoints ignored in git)
- `PINN/notebooks/` Jupyter notebooks
- `PINN/plots/` generated figures
- `PINN/src/` source code and utilities
- `heatflux/` MATLAB/solver work (ignored in git per .gitignore)
- `venv/` local Python environment (ignored in git)

## Notes
- Large data files, checkpoints, and environments are excluded via `.gitignore`.
- Add notebooks to `PINN/notebooks/`.
